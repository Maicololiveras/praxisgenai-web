Features del Dashboard

Cada feature diseñado para dar visibilidad total sobre tus agentes desplegados.

◉

Agent Overview

Grid de todos los agentes desplegados con indicadores de estado: activo, idle, error. Vista consolidada para decisiones rápidas.

✉

Live Conversation Logs

Feed en tiempo real de conversaciones de agentes con búsqueda, filtros por fecha, agente, sentimiento y cliente.

☷

Performance Metrics

Charts para response time, satisfaction score, resolution rate y cost per conversation. Comparativas por período.

☰

Usage Analytics

Conversaciones por día/semana/mes, horas pico, temas populares. Insights accionables para optimización.

⚠

Alert Management

Reglas configurables: downtime > 5min, satisfacción < 80%, presupuesto excedido. Notificaciones por email, Slack, webhook.

⚒

Multi-Tenant View

Switch entre organizaciones cliente con datos aislados. Cada tenant ve solo sus agentes, métricas y configuraciones.

⇩

Export & Reports

Descarga de reportes CSV/PDF. Reportes programados por email. Dashboards compartibles con stakeholders.

⚙

Agent Configuration

Ver/editar settings de agentes remotamente. Permisos por rol: admin full access, viewer read-only.

Integración con Agent OS

Los dos productos trabajan en sinergia: Agent OS crea y despliega, Web Platform monitorea y analiza.

Agent OS

(Desktop Application)

Crea, configura y despliega agentes IA. Entorno de desarrollo y testing local.

telemetría (gRPC/API)
config updates

Web Platform

(Dashboard & Landing)

Monitorea en tiempo real, analiza performance y muestra métricas a stakeholders.

Flujo bidireccional: Los agentes envían telemetría al dashboard via gRPC/API. El dashboard puede enviar actualizaciones de configuración de vuelta a Agent OS para ajustes remotos.
